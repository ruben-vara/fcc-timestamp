freeCodeCamp Timestamp Microservice

You can pass a string as a parameter to /api/timestamp/, and it will check if it's either a unix timestamp or a valid date string (examples: 1990-07-31, July 31, 1990).

  • If it is either, the app returns both the Unix timestamp and the UTC form of that date.
  • If it is not valid, it will return { error : "Invalid Date" }.
  • If it's empty, it return the current date in both formats.

Try it!

Visit one of these urls:

  • /api/timestamp/649382400000
  • /api/timestamp/July 31, 1990
